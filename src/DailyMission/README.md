# 외워둘 함수

---

- String to int : `Integer.parseInt("String");`
- String to long : `Long.parseLong("String");`
- String to char : `string.charAt(0);`
- String을 한 글자씩 split : `String[] s_array = s.split("");`
- char to int : `int i = char - '0';` // 아스키코드의 차이를 구한 다음, int로 자동 형변환
- 지수승 : Math.pow(a,b);
